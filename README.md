
🧮 Simple Calculator - CodVeda Level 1 Task 1

This is a basic command-line calculator built using Python. It performs four primary arithmetic operations:

- Addition
- Subtraction
- Multiplication
- Division

---

🚀 Features

- Interactive menu using loops
- Modular functions for operations
- Error handling for invalid input and division by zero
- Easy-to-understand and beginner-friendly structure

---

 🛠️ Tech Used

- Language: Python 3
- Platform: Command Line (CLI)
- IDE: Visual Studio Code

🏁 Outcome
This task helped reinforce Python basics like:
Loops
Functions
Conditional logic
Exception handling
